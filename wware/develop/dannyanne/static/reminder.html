IDをお忘れの場合

会員登録した際にダニー＆アン ( Danny&Anne ) オフィシャルストアがお知らせする「　【ダニー＆アン ( Danny&Anne ) オフィシャルストア】 会員登録のご完了　」と、ご登録いただいた「メールアドレス」が会員IDとなります。

ユーザIDを紛失した場合は、こちらの「お問い合せフォーム」からお問い合せください。 ご登録いただいているメールアドレス宛に、ユーザIDとパスワードを記載したメールをお送りいたします。

パスワードをお忘れの場合

ダニー＆アン ( Danny&Anne ) オフィシャルストアに会員登録した際、パスワード欄に任意で入力した文字が会員パスワードとなります。

パスワードを紛失した場合は、こちらの「パスワードお問い合せ画面」からお問い合せください。 ご登録いただいているメールアドレス宛に、ユーザIDとパスワードを記載したメールをお送りいたします。
