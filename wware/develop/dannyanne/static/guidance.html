SSLの設定

ダニー＆アン ( Danny&Anne ) オフィシャルストアではお買い物や会員登録などの際に入力していただくお客様の情報を守るために「SSL」という暗号化通信に対応しております。「SSL」とはインターネット上の情報を暗号化して通信するシステムです。「SSL」の利用が有効に設定されていることをご確認の上、ご利用ください。

SSLの設定方法

Windows版Internet Explorerの場合

  1. メニューの【ツール】＞【インターネットオプション】を選択してください。
  2. 【インターネットオプション】画面内の「詳細設定」を選択してください。
  3. 「セキュリティ」項目の「SSL3.0を使用する」にチェックが入っているか確認し、入ってない場合は チェックをして、「OK」ボタンをクリックしてください。

Mac版Internet Explorerの場合

  1. メニューの【編集】＞【初期設定】を選択
  2. 「Webブラウザ」-「セキュリティー　ゾーン」を開いてください。
  3. 「セキュリティー　ゾーン」を「中（安全）」に設定してください。

Netscape Navigator（6.x）の場合

  1. メニューの【編集】＞【設定】選択
  2. 【プライバシーとセキュリティ】＞【SSL】を選択
  3. 「SSLプロトコルのバージョン」項目の「SSLバージョン2をオンにする」「SSLバージョン3をオンにする」にチェックが入っているか確認して入っていない場合はチェックをし、「OK」ボタンをクリック。

クッキーの設定

ダニー＆アン ( Danny&Anne ) オフィシャルストアではクッキー（Cookie）を使用して通信の管理を行っております。
クッキーの使用が無効になっている場合やクッキーが利用できないブラウザは正しくページを表示できません。
ブラウザにてクッキーの使用許可を有効に設定していただいた上でご利用いただきますようお願いいたします。

クッキーの設定方法

Windows版Internet Explorer（6.x）の場合

  1. メニューから【ツール】＞【インターネットオプション】を選択
  2. 「プライバシー」を選択し、「詳細設定」ボタンをクリックし、「Cookie」項目の「自動Cookie処理を上書きする」にチェックをして、「ファーストパーティのCookie」「サードパーティのCookie」で「受け入れる」を選択し、「OK」ボタンをクリックしてください。

Mac版Internet Explorerの場合

  1. メニューから【編集】＞【初期設定】を選択
  2. 「受信ファイル」-「Cookie」を選択し、「Cookieを受信したとき」項目で「受け入れる」を選択し、「OK」ボタンをクリックしてください。

Netscape Navigator（6.x）の場合

  1. メニューから【編集】＞【設定】を選択
  2. 「プライバシーとセキュリティ」-「Cookie」を選択し、「Cookie」項目で、「すべてのCookieを有効にする」を選択し、「OK」ボタンをクリックしてください。

javaScriptの設定

ダニー＆アン ( Danny&Anne ) オフィシャルストアでは各ページにJavaScriptの記述を含んでいます。
JavaScriptに対応していないブラウザや設定が無効になっている場合は正しくページを表示できません。
ブラウザにてJavaScriptの利用を有効に設定してご利用ください。
また、モバイル端末などではJavaScriptに対応していない機種あります、その場合は製造元にお問い合わせください。

JavaScriptの設定方法

Windows版Internet Explorer（6.x）の場合

  1. メニューから【ツール】＞【インターネットオプション】を選択
  2. 「セキュリティ」を選択し、webコンテンツのゾーンは「インターネット」を選択し、「このゾーンのセキュリティのレベル」の中にある「レベルのカスタマイズ」ボタンをクリックしてください。
  3. 「設定」ウィンドウ内にある「スクリプト」-「Javaアプレットのスクリプト」・「アクティブスクリプト」項目を「有効にする」をチェックをして、「OK」ボタンをクリックしてください。

Mac版Internet Explorerの場合

  1. メニューから【編集】＞【初期設定】を選択
  2. 「Webブラウザ」-「Webコンテンツ」を選択して、「アクティブコンテンツ」項目の「スクリプトを有効」にチェックをして、「OK」ボタンをクリックしてください。

スタイルシートの設定

ダニー＆アン ( Danny&Anne ) オフィシャルストアでは、スタイルシート（WEB上の文字色など見栄えを揃えるもの）を利用しています。正しい情報をご覧いただくためスタイルシートを有効にしてください。

スタイルシートの設定方法

Windows版Internet Explorer（6.x）の場合

  1. メニューから【ツール】＞【インターネットオプション】を選択
  2. 「Webブラウザ」-「Webコンテンツ」を選択し「ページのコンテンツ項目」の「スタイルシートの使用（表示）」「ページ指定のフォントを使用」「ページ指定の色を使用」にチェックをして「OK」ボタンをクリックしてください。「独自のスタイルシートを使用」にチェックが入っている場合ははずしてください。

Netscape Navigator（6.x）の場合

  1. メニューから【編集】＞【初期設定】を選択
  2. 「表示」-「フォント」を選択し、「ドキュメントで他のフォントを使用できるようにする」にチェックをする
  3. さらに「配色」を選択して「Webページ指定する色と背景を常に利用」にチェックをして、「OK」ボタンをクリックしてください。

キャッシュの設定

必須ではありませんが、ダニー＆アン ( Danny&Anne ) オフィシャルストアの最新情報を閲覧していただくためにキャッシュの設定を以下のように設定していただくといつでも最新の情報がご覧いただけます。

キャッシュの設定方法

Windows版Internet Explorerの場合

  1. メニューから【ツール】＞【インターネットオプション】を選択
  2. 「インターネット一時ファイル」の「設定」ボタンをクリックし、「保存しているページの新しいバージョンの確認」項目で「ページを表示するごとに確認する」を選択し、「OK」ボタンをクリックしてください。

Mac版Internet Explorerの場合

  1. メニューから【編集】＞【初期設定】を選択
  2. 「Webブラウザ」-「詳細設定」を選択して、「キャッシュ」項目の「ページの更新」で「常に」を選択して、「OK」ボタンをクリックしてください。

Windows版Netscape Navigatorの場合

  1. メニューから【編集】＞【設定】を選択
  2. 「詳細」-「キャッシュ」を選択して、「キャッシュしたドキュメントとネットワーク上のドキュメントとの比較」項目で「毎回」を選択して、「OK」ボタンをクリックしてください。

Mac版Netscape Navigatorの場合

  1. メニューから【編集】＞【設定】を選択
  2. 「詳細」-「キャッシュ」を選択して、「キャッシュしたページとネットワーク上のページとの「比較」項目で「毎回」を選択して、「OK」ボタンをクリックしてください。
