・All in one Japanize package for Japanese users / 日本語化エクステンション（全部入り）
　http://www.magentocommerce.com/magento-connect/magento-ec-9160.html
　弊社製日本語化エクステンションが全て揃う便利パッケージです。

・Price Display Format / Magento価格表示調整エクステンション（Ver.0.9.16 以上）
　http://www.magentocommerce.com/magento-connect/catalog/product/view/id/11946/
　円建て決済を前提としておりますので、価格に小数点以下の数値が含まれているとエラーとなります。
　このエクステンションを組み合わせるとその問題を解決できます。
　エクステンション導入後、
　「システム」→「設定」→「通貨セットアップ」→「通貨表示オプション」→「精度」を0に設定します。


■オーダーIDについて
　Magento側で発行した注文番号は、GMO様の管理画面「PGマルチペイメントサービス」の
　取引検索等で取引の情報を参照した場合、「オーダーID」という項目のハイフンの右側が対応しております。


■index.phpの書き換えについて
　本エクステンションのGMOモジュールインストール機能を使用した場合、自動的に
　　[Magentoインストールディレクトリ]/index.php
　の一部内容を書き換えます。

　書き換える前のファイルは、
　　index_php.backup.[バックアップ日時]
　というファイル名でバックアップされます。

　＜変更される箇所＞
　error_reporting(E_ALL | E_STRICT);
　　　　　　　↓
　error_reporting((E_ALL ^ E_STRICT) && (E_ALL ^ E_DEPRECATED));


--------------------------------------------------------------------------------
セットアップ
--------------------------------------------------------------------------------
Magento Connectから本エクステンションを導入した後、以下の手順でエクステンションの
セットアップを行ってください。

■「GMOモジュール」を本モジュールのインストーラーを使用してインストールする
　本エクステンションには、GMO様から提供された「モジュール」をインストールする機能が搭載されています。
　※GMO様のモジュールが無いと本エクステンションは動作しません。

　[手順]
　１．GMO様のサイトからGMOモジュールファイル（Zip形式）をダウンロード
　２．「Magento Admin Panel」にアクセス
　３．「システム」「設定」「GMO Payment Module」をクリック
　４．「GMOモジュール (Zip圧縮ファイル)」欄の参照ボタンをクリック
　５．GMOモジュールファイル（Zip形式）を選択
　６．「設定を保存」ボタンをクリック
　　　※unzipがインストールされていない場合エラーになりますので、unzipを予めインストールしておいてください。
　　　アップロードが正常に終了すると、「Infomation」欄にインストールされたパス情報が表示されます。
　引き続き、コンフィグファイル（２種）をインストールします。
　７．GMOモジュールファイル（Zip）を解凍する
　８．「GMOコンフィグファイル (connector.properties)」欄には、connector.propertiesファイルを、
　　　「GMOログ設定ファイル (log.properties)」欄には、log.propertiesファイルを、それぞれの欄の参照ボタンを
　　　クリックして選択してください。
　　　それぞれのファイルは、解凍したファイルの中の以下の位置にあります。
　　　　PHP/gpay_client/src/conf/
　　　connector.propertiesファイルは、テスト環境用と本番環境用の2種類があります。
　　　実行するGMO様接続環境に合わせて選択してください。
　９．「設定を保存」ボタンをクリック

　コンフィグファイルのみを入れ替える場合は、上記８～９を実行します。
